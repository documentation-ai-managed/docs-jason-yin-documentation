## Overview

Our investment process is designed to identify and nurture promising Web3 projects led by student entrepreneurs. We focus on thorough evaluation, mentorship, and strategic funding to ensure long-term success.

## Evaluation Criteria

| Criterion | Weight | Description |
|-----------|--------|-------------|
| Innovation | 30% | Uniqueness of Web3 solution |
| Team | 25% | Experience and commitment |
| Market | 25% | Size and accessibility |
| Traction | 20% | User growth and validation |

<CodeGroup tabs="TypeScript,Go">
  ```typescript
  interface ProjectEvaluation {
    innovation: number;
    team: number;
    market: number;
    traction: number;
  }
  
  const evaluateProject = (project: ProjectEvaluation): number => {
    return (project.innovation * 0.3) + (project.team * 0.25) + 
           (project.market * 0.25) + (project.traction * 0.2);
  };
  ```
  ```go
  type ProjectEvaluation struct {
      Innovation float64
      Team       float64
      Market     float64
      Traction   float64
  }
  
  func evaluateProject(p ProjectEvaluation) float64 {
      return (p.Innovation * 0.3) + (p.Team * 0.25) + 
             (p.Market * 0.25) + (p.Traction * 0.2)
  }
  ```
</CodeGroup>
